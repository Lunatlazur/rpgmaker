# TachiE
RPGツクールMV用 立ち絵プラグイン

<!-- % プラグインのスクリーンショット % -->

---

このプラグインはキャラクターの立ち絵を表示できるようにします。

---

# 導入方法
`build` フォルダ内の `Lunatlazur_Tachie.js` を導入したいRPGツクールMVのプロジェクト内の `js/plugins` フォルダにコピーしてください。
