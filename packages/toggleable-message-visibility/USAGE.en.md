Usage
=====

Plug-in command list
====================

SHOW_MESSAGE
------------

    SHOW_MESSAGE

Display the message window.


HIDE_MESSAGE
------------

    HIDE_MESSAGE

Hide the message window.


TOGGLE_MESSAGE
--------------

    TOGGLE_MESSAGE

Toggles the display status of the message window.  
If it is displayed, it is hidden.  
If it is hidden, it is displayed.
