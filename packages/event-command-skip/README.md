# EventCommandSkip
RPGツクールMV用 イベントコマンドスキッププラグイン

![スクリーンショット](./doc/event-command-skip.png)

---

このプラグインはCtrlキーを押すことでイベントコマンドをスキップ
できるようにします。

---

# 導入方法
`build` フォルダ内の `Lunatlazur_EventCommandSkip.js` を導入したいRPGツクールMVのプロジェクト内の `js/plugins` フォルダにコピーしてください。
