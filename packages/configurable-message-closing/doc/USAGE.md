プラグインコマンド一覧
======================

メッセージの表示を継続
----------------------

    メッセージの表示を継続 <継続有無>

メッセージウインドウを自動で閉じるかどうかを切り替えます。


    <継続有無> { ON | OFF }

ON でメッセージウィンドウを自動で閉じないようにします。  
OFF にしたときにメッセージが存在しなければ、メッセージウィンドウは即座に閉じ
られます。


呼び出し例

    メッセージの表示を継続 ON

メッセージウィンドウを自動で閉じないようにします。


メッセージを閉じる
------------------

    メッセージを閉じる

メッセージウィンドウを閉じます。
